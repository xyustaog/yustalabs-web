Lógica Compartida
Patrones de código reutilizados en todas las apps

🗄️ Storage Pattern

Todas las apps demo usan localStorage como base de datos. La clave se define en storageKey (ej: as-v1). Se lee con JSON.parse(localStorage.getItem(key)) y escribe con localStorage.setItem(key, JSON.stringify(db)). Aura Commerce es la única excepción con Firestore.

🖨️ Print Pattern

Las funciones de impresión abren window.open() con HTML+CSS completo, llaman w.print() y w.close(). El CSS incluye @media print para ocultar botones y aplicar márgenes de papel.

📱 WhatsApp Pattern

Links generados como https://wa.me/549XXXXXXXXXX?text=... con encodeURIComponent() para el mensaje. En batch, se abren múltiples tabs o se muestra modal con links individuales.

📊 SVG Chart Pattern

Gráficos de barras sin librerías externas. Se calcula maxVal, cada barra es un <rect> con height = (val/maxVal) * chartH. Labels con <text>. Usado en Fit, Finance, Food.

🔒 Delete Validation Pattern

Antes de eliminar un registro maestro, se verifica que no existan registros dependientes. Ejemplo: db.turnos.some(t => t.profId === id). Si existen, se muestra alert() con conteo y se retorna sin eliminar.

💰 Precio Snapshot Pattern

Al crear un ítem transaccional (comanda, venta), se copia el precio actual al registro: precio_cobrado = menu[id].precio. Los totales siempre leen precio_cobrado, no el precio actual del catálogo. Implementado en Food y Commerce.

🎨 Design System

Fondo: #080810 · Cards: #0d0d1a · Border: rgba(255,255,255,.08) · Font: Inter · Accent: por app (ver colores). Botones primarios con color de acento. Modales: position:fixed backdrop oscuro.

🌐 Deploy Stack

Todas las apps demo en Vercel como sitios estáticos (un solo index.html). Landing pages en yustalabs.com (yustalabs-web). Demos accesibles en yustalabs.com/aura-[producto]/demo vía iframe con topbar branded.

Agentes IA
Framework de agentes autónomos compartido por todas las apps Aura
FRAMEWORK

Patrón de 3 fases

Cada agente Aura se construye sobre el mismo ciclo: Detecta un trigger (evento, fecha, umbral) → Analiza cruzando data del negocio (Firestore, historial, reglas) → Actúa ejecutando una acción automática (WhatsApp, email, notificación, alta de registro). Los 6 arquetipos se adaptan al rubro de cada app, pero la arquitectura y el runtime son compartidos.

💬
El Recepcionista / Vendedor Virtual
Agente de atención
Atención 24/7 por WhatsApp. Responde consultas, toma reservas, envía catálogo, califica leads y deriva a humano cuando corresponde.
DetectaMensaje entrante de WhatsApp o formulario web sin respuesta humana
AnalizaIntención del cliente + disponibilidad (agenda/stock) + historial previo
ActúaResponde, agenda turno, envía link de pago o escala a dueño
💸
El Cobrador
Agente de cobranzas
Persigue pagos y cuotas vencidas sin intervención humana, escalando tono y canal según días de atraso.
DetectaCuota, factura o plan próximo a vencer / ya vencido
AnalizaMonto, días de atraso, historial de pago, método preferido del cliente
ActúaEnvía WhatsApp con link de pago, recordatorio escalado y alerta al dueño
📦
El Almacenero / Stock Manager
Agente de inventario
Monitorea inventario en tiempo real, anticipa quiebres y genera órdenes de reposición sugeridas al proveedor.
DetectaProducto debajo de umbral mínimo o velocidad de venta anómala
AnalizaStock actual + ventas últimos 30 días + lead time del proveedor
ActúaGenera orden de compra sugerida y notifica al encargado por WhatsApp
📊
El Analista
Agente de reportes
Resume el negocio en lenguaje natural. Entrega reportes ejecutivos diarios/semanales con insights accionables.
DetectaCorte de día/semana/mes o pedido manual del dueño
AnalizaKPIs (ventas, ticket promedio, cancelaciones, márgenes) vs. período anterior
ActúaEnvía resumen por WhatsApp/email con 3 insights y acción recomendada
📣
El Promotor / Community
Agente de marketing
Lanza campañas y recordatorios segmentados. Genera copy, arma lista y dispara en batch por WhatsApp.
DetectaFecha clave (cumple, temporada, feriado) o caída en métrica (ocupación, ventas)
AnalizaSegmento objetivo + oferta disponible + historial de respuesta previa
ActúaGenera mensaje personalizado y envía batch WhatsApp con tracking
❤️
El Fidelizador
Agente de retención
Detecta clientes inactivos antes de perderlos y dispara acciones de reenganche con incentivo personalizado.
DetectaCliente sin interacción/compra en X días según rubro
AnalizaLTV, frecuencia previa, productos/servicios favoritos del cliente
ActúaEnvía WhatsApp con beneficio puntual (descuento, regalo, turno prioritario)
